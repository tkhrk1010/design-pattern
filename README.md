# Why it exists?

# How to use
1. Attach VSCode to container (HOME: /usr/src/)
2. cd directory
3. compile .java
4. execute java
   
ex)  
    $ cd FaFactoryMethod/Sample  
    $ javac Main.java
    $ java Main
